栅格系统

Bootstrap 提供了一套响应式、移动设备优先的流式栅格系统，随着屏幕或视口（viewport）尺寸的增加，系统会自动分为最多12列。它包含了易于使用的预定义类，还有强大的mixin 用于生成更具语义的布局。

  • "行(row)"必须包含在.container(固定宽度)或.container-fluid（100% 宽度）中，以便为其赋予合适的排列（aligment）和内补（padding）。
  • 通过"行(row)"在水平方向创建一组"列(columm)"
  • 栅格系统中列是通过指定1-12的值来表示其跨越的范围。列如，三个等宽的列可以使用三个.col-xs-4来创建
  • 如果一“行（row）”中包含了的“列（column）”大于 12，多余的“列（column）”所在的元素将被作为一个整体另起一行排列。

中心内容

通过添加.lead类可以让段落突出显示
  • 对于被删除的文本使用< del>标签
  • 无用的文本使用< s>标签
  • 额外插入的文本使用< ins>标签
  • 带下滑线的文本，使用< u>标签
  • 小号文本是父容器大小的85%。你还可以为行内元素赋予.small类以代替任何< small>元素
  • 斜体< em>
  • 缩略语：当鼠标悬停在缩写和缩写词上时就会显示完整内容< abbr>标签，人名居多

状态栏

通过这些状态类可以为行或单元格设置颜色。

Class 描述
.active 鼠标悬停在行或单元格上时所设置的颜色
.success 标识成功或积极的动作绿色
.info 标识普通的提示信息或动作蓝色
.warning 标识警告或需要用户注意橙色
.danger 标识危险或潜在的带来负面影响的动作红色

响应式表格

将任何.table元素包裹在.table-responsive元素内，即可创建响应式表格，其会在小屏幕设备上(小于768px)水平滚动，当屏幕大于 768px 宽度时，水平滚动条消失。

	 		<div class="table-responsive">
	 		<tableclass="table">
	 		 		...
	 		 		< /table>
	 		 		< /div>
