1. DOM Document 对象

回到首页

1.1. Document 对象是 Window 对象的一部分，可通过 window.document 属性对其进行访问。

2. Document 对象方法

方法 描述
close() 关闭用 document.open() 方法打开的输出流，并显示选定的数据。
getElementById() 返回对拥有指定 id 的第一个对象的引用。
getElementsByTagName() 返回带有指定标签名的对象集合。。
getElementsByName() 返回带有指定名称的对象集合。
open() 打开一个流，以收集来自任何 document.write() 或 document.writeln() 方法的输出。
write() 向文档写 HTML 表达式 或 JavaScript 代码
writeln() 等同于 write() 方法，不同的是在每个表达式之后写一个换行符。

2.创建节点

nodeType 属性返回数值指定节点类型
如果节点是元素节点 则nodeType属性返回1
如果节点是属性节点 则nodeType属性返回2
childNodes 表示对象的所有子节点的列表 返回的值可以看这是一个数值 它具有length属性 element.style设置或返回的style属性

方法 描述
appendchild() 把新的子节点添加到指定节点
removechild() 删除子节点
replacechild() 替换子节点
insertBefore() 在指定的子节点前面插入新的子节点
createAttribute() 创建属性节点
createElement() 创建元素节点
createTextNode() 创建文本节点
getAttribute() 返回指定属性值
setAttribute() 指定属性设置或修改指定的值
在本例中，当用户在 < h1> 元素上点击时，会改变其内容：
		   < h1 onclick="this.innerHTML='谢谢!'">请点击该文本

THML DOM 节点

    HTML 文档中的所有内容都是节点：

  • 整个文档是一个文档节点
  • 每个 HTML 元素是元素节点
  • HTML 元素内的文本是文本节点
  • 每个 HTML 属性是属性节点
  • 注释是注释节点
